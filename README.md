## Run Locally

Clone the project

```bash
  git clone https://github.com/vitamins999/portfolio.git
```

Go to the project directory

```bash
  cd portfolio
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
